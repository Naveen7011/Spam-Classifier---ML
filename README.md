Spam Classifier

This project is a Machine Learning based Spam Classifier that analyzes text messages and predicts whether a message is Spam 🚫 or Ham ✔️ using Natural Language Processing (NLP) techniques.

Features

💠 Detect Spam or Ham messages
💠 Single Message Prediction
💠 Bulk Message Prediction using CSV
💠 Interactive Streamlit web interface

Technologies Used

Python • Pandas • NumPy • Scikit-learn • NLP • Streamlit

Use Case

Helps automatically filter spam messages and identify genuine messages.
